Comparaison d’approches algorithmiques
Implémentation d’un besoin client (2022-2023)

  • Langage : Java
  • Durée : 4 jours

L'objectif du projet est de créer un de classification automatique de dépêches aussi fiable et aussi rapide que possible. Une dépêche est un court texte correspondant à une information journalistique, telle que :

Coupe de l'UEFA : l'OM victorieux, les Girondins défaits L'OM a remporté son match à domicile sur le score de 3 à 0. Bordeaux a perdu dans les arrêts de jeu.

Le programme doit être capable de classer automatiquement une dépêche dans une des catégories suivantes :

  • Politique
  • Sport
  • Sciences
  • Divertissement
  • Autres
    •

La tâche principale consiste à classer des documents dans différentes catégories en se basant sur les mots qui indiquent leur appartenance à une catégorie spécifique. Pour cela, nous créons des lexiques pour chaque catégorie, composés de mots "marqueurs" et associés à des poids. Les poids représentent le degré avec lequel chaque mot indique l'appartenance à une catégorie, variant de 1 (faible indication) à 3 (forte indication).

À travers ce projet, j'ai pu

  • Parfaire ma pratique de Java
  • Apprendre à utiliser des algorithmes de tri
  • Utiliser Java dans un contexte donné
